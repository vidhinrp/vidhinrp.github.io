Introduction¶

This data was scraped from the 311 Service Request data available from the Charlotte Open Data Portal. The initial csv contained data from 2017 to 2020 and was filtered to only include December, 2018. A variety of questions are explored in this analysis including:

  • What complaints happened every day?
  • Which hour had the most requests?
  • Which requests happened after hours?

def averageNumberRequestsPerHour(calls):
    ''' Return the average number of requests per hour '''
    per_hour = ((calls.loc[calls.index[-1],'Hour']+1)*(calls.loc[calls.index[-1],'Day']+1))

def compaintsEachDay(calls):
    ''' Return an alphabetical list of complaints that occur every day in the data'''
    div = list(calls.REQUEST_TYPE.unique())
    for day in range(np.max(calls.Day)+1):
        for x in div:
            if x not in calls.loc[calls['Day'] == day, 'REQUEST_TYPE'].unique():
                div.remove(x)
    return(sorted(div))
 
    
print('Complaints that happened every day', compaintsEachDay(calls))

8. How long can I sleep?¶

The function longestTimeWithNoRequests returns the length of the longest run of hours with no requests having the complaint string anywhere in the complaint.

In [106]:
def longestTimeWithNoRequests(word, calls):
    ''' Return the number of hours with no requests having the parameter
    request string anywhere in the request. '''
    max_hours = []
    for x in calls.REQUEST_TYPE:
        if word in x:
            max_hours.append(True)
        else:
            max_hours.append(False)

    return (np.diff(calls.Hour[max_hours]).max()-1)

Word = 'RECYCL' # anywhere in the Complaint
print('Longest run of quiet is', longestTimeWithNoRequests(Word, calls), 'hours.')

def averageNumberEachDayOfWeek(zipcode, calls):
    ''' Return an array of 7 entries that is the average number of requests with zipcode, 
    for each day of the week. '''
    num_days = (calls.groupby('Day_Week')['Day'].nunique())            
    description = (calls.loc[calls.ZIP_CODE==zipcode, 'Day_Week'].value_counts())
    return(np.array(description/num_days))

zipcode = 28277
print('Average complaints each day of the week in',zipcode,'is',
      averageNumberEachDayOfWeek(zipcode, calls))

10. How far will an inspector have to travel?¶

The function distanceTraveled returns how far will an inspector have to travel to visit, in the order they occurred, all of the coordinates given complaint and city.

In [136]:
def distanceTraveled(complaint, city, calls):
    ''' Return the distnace traveled for an inspector to travel and visit each
    complaint in city.'''
    coord = calls[(calls.REQUEST_TYPE == complaint)&(calls.CITY == city)][['X_COORD','Y_COORD']].values
    
    sumcoord = []
    for x in range(len(coord)-1):
        sumcoord.append(np.sqrt(np.sum(np.square([coord[x+1]-coord[x]]))))
        
    return(np.sum(sumcoord))
    
City = 'CHARLOTTE'
Complaint = '311 DOCUMENT'
print('Distance traveled to go for each', Complaint, 'in', City, 'is',
      distanceTraveled(Complaint, City, calls))

def descriptorsAfterHours(calls):
    ''' Return the unique, request descriptors, in alphabetical order that came in
    only while the office was closed but never while it was open.'''
    after_hours = calls[~(calls.Hour>=6) & (calls.Hour<=18) & (calls.Day_Week>=0) & (calls.Day_Week<=4)]
    return(after_hours['REQUEST_TYPE'].unique())
            

print('After hours requests are', descriptorsAfterHours(calls))

Conclusion¶

311 requests within the Charlotte region in 2018 encompassed many issues. Findings from this ESDA included:

  • Requests were more likely to be filed in during the weekdays than the weekends. However, that doesn't mean that requests weren't filed during the weekends as the longest period of time with no requests was a short 9 hours.

  • The busiest day for the 311 response team was on December 3rd. More specifically, 3PM on December 3rd saw the most complaints filed. Coincidentally, December 3rd 2018 was also Charlotte's 250th Anniversary.

  • Intrestingly, a complaint relating to flooding was filed everyday in 2018. It would be worthwhile to dig further into where these flooding requests came from and what type of flooding was complained about (i.e. natural, faulty plumbing) as flooding seems like an unlikely complaint to be occuring frequently.
