Below are some data science/analysis projects I've recently worked on.

Extracting Patient Data: Using synthetic patient data from the Synthea database that was adopted to fit the OMOP schema, I created PySpark scripts to extract relevant patient information. The scripts extract data relating to demographic information, medical measurements, and medical conditions for patients based on recorded visits. Each patient and visit has an unique ID. I created some sample conditions to demonstrate how the scripts work and generated basic plots to visualize the data. This can be viewed by clicking on the link above. The scripts can be accessed here. Libraries/Skills Demonstrated: Pandas, Seaborn, PySpark.

Predicting Hospital Admission Based on Emergency Admission Triage: I built a XGBoost(AUC = 0.89) and a Logistic Regression (AUC = 0.77) model using sklearn to predict hospital admission based on emergency admission triage using a dataset with over 500,000 entries and 900 features. I conducted model evaluation and interpretability using SHAP plots, some of which are included in the linked notebook. Libraries/Skills Demonstrated: Pandas, Seaborn, Numpy, Scikit-Learn, XGBoost, SHAP Plots.

Progress made on Sustainable Development Goal 7.3: In 2015, the United Nations set a series of 17 interconnected development goals for the world called the "Sustainable Development Goals" (SDG). Using data from the Global SDG Indicators Database, I created a multi-page dashboard on Tableau. The dashboard focuses on one specific target of the SDGs focused on improving energy efficiency. Preliminary data cleaning and transforming was done in MySQL, while any necessary modifications during dashboard building were done using Tableau as a means to test how far the tool can go. Libraries/Skills Demonstrated: MySQL, Tableau.

Building and Querying a World Cup Database: Using data from the 2014 and 2018 World Cups, I built a database using Bash and PostgreSQL. Then, using Bash scripts, I queried the database to find out some basic information about the games played. Libraries/Skills Demonstrated: Bash, PostgreSQL.

Water Wars: Using data from the Pacific Institute, I conducted analysis on armed conflicts associated with water resources and water systems, as identified from news reports, eyewitness accounts, and other conflict databases. Libraries/Skills Demonstrated: Pandas, Seaborn, GeoPandas, Text Mining.
