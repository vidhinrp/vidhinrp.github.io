Issues fought over¶

Text Mining¶

Let's take a look at what issues have influenced conflicts through the course of time. To do this, we will use the Description column from conflicts and identify the most frequent keywords by Start year.

First, we will go through and tokenized words for each entry within Description and remove any commonly used words (i.e. a, an, the).

In [30]:
from nltk.corpus import stopwords
from nltk import word_tokenize
import string
stoplist = stopwords.words('english') + list(string.punctuation)
stoplist = set(stoplist)
In [31]:
def preprocess(text):
    return [word for word in word_tokenize(text) if word.lower() not in stoplist and not word.isdigit()]
In [32]:
conflict['Filtered_Description'] = conflict['Description'].apply(preprocess)
conflict['Word_Frq'] = conflict['Filtered_Description']

Now, we will count the frequency of each word in this column by year and export it to arr.

In [33]:
def unnesting(df, explode):
    idx=df.index.repeat(df[explode[0]].str.len())
    df1=pd.concat([pd.DataFrame({x:np.concatenate(df[x].values)} )for x in explode],axis=1)
    df1.index=idx
    return df1.join(df.drop(explode,1),how='left')
In [34]:
arr = unnesting(conflict,['Word_Frq']).groupby(['Start','Word_Frq']).size().sort_values().reset_index()
arr = arr.rename(columns = {0:"Freq", "Word_Frq": "Word"})

We can now take arr and group it further to see the word with the maximum frequency in each year.
